# ⚡ Fitness Coach PWA

> A unified, cross-platform fitness experience. This Progressive Web Application (PWA) serves as a digital fitness coach, offering a seamless and identical UI/UX across desktop and mobile devices. Built with React, TypeScript, and Firebase, it features real-time workout tracking and secure user authentication.

---

## 🚀 Getting Started

> Clone the Repository:

    git clone https://github.com/natalijakostadinoska/Fitness-App.git
    cd Fitness-App

---

## Requirements

> To run the project on your Visual Studio Code program, you'll need to:

    npm install # to install the dependencies

> Environment Setup: Enable Authentication (Email/Password) in the Firebase Console. Create a Firestore Database in Test Mode. Ensure your src/firebase.ts is updated with your configuration keys.

> Then:

    npm run dev # to run the whole application

> The app will be available at http://localhost:3000

---

## 🛠️ Tech Stack

> Frontend: React.js, TypeScript, Tailwind CSS
> Backend: Firebase (Authentication & Firestore)
> Notifications: React-Toastify
> Build Tool: Vite

---

## ✨ Key Features

> Cross-Platform Parity: Fully responsive PWA design that looks and feels native on any device.
> Secure Auth: User registration and login powered by Firebase with professional error handling.
> Workout Tracking: Record activities instantly and sync them to a personal history log.
> Real-time Data: Instant UI updates using Firestore listeners.
> Modern UI: Clean, high-fidelity design with custom toast notifications.
